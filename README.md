# PB Acrylic

Premium custom acrylic products — signage, displays, awards, and more.

**Live site:** [https://fit-pikaboo.github.io/pb-acrylic-wb/](https://fit-pikaboo.github.io/pb-acrylic-wb/)

## About

PB Acrylic is a digital showroom showcasing our range of precision-crafted acrylic products. Browse our catalog, explore customization options, and view our past works.

## Tech Stack

- Vanilla HTML, CSS, JavaScript
- Tailwind CSS (via CDN)
- No build tools or backend required

## Local Development

Open `index.html` in a browser, or run a local server:

```bash
npx serve .
```

## Deployment

Static files — deploy anywhere. Currently hosted on GitHub Pages.
